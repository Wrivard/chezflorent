## 🎯 Contexte du projet

Tu vas construire un site web vitrine **haut de gamme, digne d'une grande agence créative** pour un restaurant québécois nommé **Chez Florent**. C'est un bistro-bar chaleureux et accessible, avec un ton ludique et un vrai caractère artisanal — pensez ardoise du jour, produits du terroir, plats à partager avec personnalité (pizzas signatures, charcuteries, sandwichs créatifs).

**Le site doit avoir l'air d'avoir été conçu par une agence de design primée** — pas un template restaurant générique. Référence-toi à des sites comme ceux qu'on retrouve sur Awwwards, SiteInspire ou The FWA dans la catégorie restaurant/hospitality. On vise l'éditorial, le sophistiqué, avec une touche de nonchalance.

**Stack technique :**
- HTML/CSS/JavaScript vanilla (pas de framework lourd — performance et simplicité)
- Une seule page (one-page scroll) avec navigation ancrée (anchor links) vers chaque section
- Site 100% statique, déployable tel quel
- Responsive mobile-first, doit être impeccable sur mobile, tablette et desktop
- Animations subtiles au scroll (Intersection Observer), pas de librairies externes lourdes

**Sections à inclure (dans l'ordre) :**
1. **Hero / Accueil** — impact visuel fort, le logo, une accroche éditoriale
2. **À propos courte** — 2-3 phrases qui plantent l'identité (intégrée dans la home)
3. **Menu / L'ardoise** — présentation soignée du menu (voir contenu plus bas)
4. **Réservation** — formulaire élégant
5. **Contact** — adresse, heures, téléphone, carte (placeholder), réseaux sociaux
6. **Footer** — minimal et sobre

## 🎨 Brand Guide

### Palette de couleurs

Inspirée du logo (fond vert sapin très foncé, lettrage crème pêche, ombres orange brûlé) :

```css
--color-bg-primary: #0E1F1C;        /* Vert sapin profond — fond principal */
--color-bg-secondary: #142826;      /* Vert sapin légèrement plus clair — sections alternées */
--color-bg-tertiary: #1A332F;       /* Vert sapin élevé — cartes, encarts */
--color-cream: #F4C9A0;             /* Crème pêche — typographie principale, logo */
--color-cream-soft: #EBC097;        /* Crème adoucie — corps de texte */
--color-orange: #D85A2C;            /* Orange brûlé — accents, CTA, ombres */
--color-orange-dark: #B8451F;       /* Orange foncé — hover states */
--color-border: rgba(244, 201, 160, 0.12);  /* Bordures subtiles crème */
--color-border-strong: rgba(244, 201, 160, 0.25);
```

**Règle d'or des couleurs :**
- Le **vert sapin** domine (70% de la surface)
- Le **crème** est la couleur typographique principale (25%)
- L'**orange brûlé** est utilisé avec parcimonie pour les accents (5% — CTAs, soulignements, prix, détails graphiques)
- **Jamais** de blanc pur, **jamais** de noir pur

### Typographie

**Hiérarchie à 3 polices, toutes via Google Fonts :**

```css
/* H1 — Display, scriptée, comme le logo */
font-family: 'Caveat Brush', cursive;  /* OU 'Pacifico' en fallback proche du logo */
/* Utilisé UNIQUEMENT pour : logo wordmark, titres de sections principales (Menu, Réservation, Contact) */

/* H2-H3 — Editorial serif */
font-family: 'Fraunces', serif;
font-weight: 300, 400, 600;
/* Utilisé pour : sous-titres, noms de plats, titres secondaires */
/* Privilégier les variations italiques pour les accents éditoriaux */

/* Body — Sans-serif moderne */
font-family: 'Inter', sans-serif;
font-weight: 300, 400, 500;
/* Utilisé pour : corps de texte, navigation, descriptions, prix, boutons */
/* letter-spacing: -0.01em pour le corps, +0.15em uppercase pour les eyebrow labels */
```

**Échelle typographique (desktop) :**
- H1 hero : `clamp(4rem, 10vw, 9rem)` — Caveat Brush
- H2 section : `clamp(2.5rem, 6vw, 5rem)` — Caveat Brush ou Fraunces italic
- H3 plat : `1.75rem` — Fraunces 400
- Body : `1.0625rem` — Inter 400, line-height 1.7
- Eyebrow label : `0.75rem` — Inter 500, uppercase, letter-spacing +0.2em
- Prix : `1.25rem` — Fraunces 600

### Identité visuelle & règles de design

**Composition :**
- **Beaucoup d'espace blanc** (enfin, "espace vert" ici). Padding généreux : sections à `padding: 8rem 0` minimum sur desktop
- Layout **asymétrique** par moments (à la éditorial magazine), pas tout centré
- **Grilles brisées** : un titre à gauche, une description décalée à droite avec offset
- **Filets de séparation** très fins (`1px solid rgba(244,201,160,0.12)`) plutôt que des cards lourdes
- **Numérotation** discrète des sections en haut (01 — L'ARDOISE, 02 — RÉSERVATION) en eyebrow style

**Détails graphiques :**
- Petits ornements typographiques en orange (asterisques, slashes, tirets longs em-dash)
- Lignes verticales fines en accent
- Effet de **soulignement orange** subtil sous certains mots-clés (effet logo)
- **Pas d'icônes Material/Feather classiques** — privilégier le texte ou des SVG custom minimaux

**Animations (subtiles, jamais flashy) :**
- Fade-in + translateY(20px) au scroll via Intersection Observer
- Hover sur liens : underline qui s'étire de gauche à droite (transition 400ms ease-out)
- Hover sur boutons : background color shift + léger scale(1.02)
- Pas de parallax lourd, pas de scroll-jacking

**Coins & rayons :**
- `border-radius: 0` sur les cards et sections (look éditorial, tranchant)
- `border-radius: 2px` sur les boutons et inputs (à peine perceptible)
- Photos en `border-radius: 0` aussi — on est sur du éditorial
