You are a senior frontend engineer specialized in responsive design and UI debugging.

I have an existing website that is already designed for desktop.
Your task is to audit and fix ONLY mobile responsiveness issues while ensuring the desktop layout remains completely unchanged.

🎯 Objective

Make the website fully usable and visually correct on mobile devices (phones and small tablets) without altering desktop behavior or design.

🚫 Hard Constraints
Do NOT change desktop layout, spacing, or styling
Do NOT redesign the website
Do NOT refactor unnecessary code
Only apply mobile-specific fixes
All changes must be scoped using media queries (@media (max-width: ...))
Avoid breaking existing components
📱 Mobile Targets to Fix

Ensure perfect rendering on:

320px (small phones)
375px (iPhone SE / standard)
390–430px (modern iPhones)
768px (tablet portrait)
🔍 What to Check and Fix

Fix only if issues exist:

Layout issues
Elements overflowing horizontally
Broken grid/flex layouts on small screens
Sections not stacking properly
Typography
Text too small or too large on mobile
Line height issues affecting readability
Navigation
Mobile menu not usable or not visible
Buttons too close or too small to tap
Media (images/videos)
Images not scaling properly
Cropped or stretched media
Background images not centered or responsive
Spacing
Excessive padding/margin on mobile
Content too compressed or too spaced out
UX issues
Key content not visible above the fold
Horizontal scroll appearing anywhere (must eliminate)
🧠 Required Approach
First analyze the full codebase and identify mobile-only issues
Do NOT touch desktop styles unless absolutely necessary
Apply fixes using:
CSS media queries
minor structural tweaks only if required
flex/grid adjustments for mobile stacking
Keep changes minimal and surgical
📦 Output Requirements

After applying fixes:

Summarize what was changed
List each mobile issue found and how it was fixed
Confirm that desktop layout remains unaffected
Ensure no horizontal scrolling exists on any page
💡 Important Mindset

Treat this like a production hotfix pass, not a redesign.
Prioritize stability, consistency, and preserving the original desktop design.
